El **Sistema de Préstamos de Portátiles** nace como respuesta a la necesidad de digitalizar y optimizar la gestión de recursos tecnológicos en entornos educativos. Este proyecto busca resolver los cuellos de botella administrativos y proporcionar una experiencia de usuario ágil y transparente.

### Propósito del Sistema
El objetivo principal es centralizar la gestión de dispositivos, garantizando que el inventario esté siempre actualizado y que el proceso de solicitud sea eficiente tanto para alumnos como para profesores.

<Card title="Visión General" icon="information">
  Este sistema no solo registra préstamos, sino que implementa una lógica de negocio robusta para evitar conflictos de disponibilidad y asegurar el mantenimiento adecuado de los equipos.
</Card>

### Arquitectura de Alto Nivel
El sistema se apoya en una arquitectura de microservicios y despliegue en la nube para garantizar alta disponibilidad.



---

### Beneficios clave
* **Automatización:** Eliminación de registros en papel y hojas de cálculo manuales.
* **Trazabilidad:** Histórico detallado de cada dispositivo y usuario.
* **Accesibilidad:** Interfaz web diseñada para ser utilizada desde cualquier dispositivo.
