Esta guía detalla los pasos necesarios para desplegar y ejecutar el proyecto en un entorno local, permitiendo la evaluación y pruebas del sistema.

## Requisitos previos

* **Node.js**: Versión 18 o superior.
* **Git**: Para la gestión de versiones.
* **npm**: Incluido con Node.js.

## Configuración del backend y frontend

<Steps>

1. Acceder a la carpeta del backend  
         ```bash
      cd ../prestamosportatiles

2. Ejecutar el backend  
         ```bash
      ./gradlew bootRun  
         
3. Acceder a la carpeta del frontend  
         ```bash
      cd ../prestamos-frontend 

4. Instalar dependencias   
         ```bash
      npm install

5. Ejecutar el servidor de desarrollo  
         ```bash
      npm run dev
</Steps>

## Acceso a la aplicación

Una vez iniciado el servidor, abre el navegador y accede a:

```txt
http://localhost:4321
```

<Aside type="tip" title="Consejo">
Si el puerto 4321 está ocupado, Astro utilizará automáticamente otro puerto disponible.
</Aside>
