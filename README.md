## 📌 Introducción

### Descripción del proyecto
Aplicación web para la gestión de préstamos de portátiles en un entorno educativo. Permite iniciar sesión, solicitar un portátil disponible, devolverlo y consultar su estado.

### Justificación
Este proyecto surge para digitalizar y automatizar la gestión manual de préstamos de dispositivos en centros educativos, mejorando el control y reduciendo errores.

### Objetivos
- Automatizar el préstamo de portátiles  
- Controlar la disponibilidad de equipos  
- Registrar historial de préstamos  
- Implementar autenticación de usuarios  

### Motivación
Desarrollar una aplicación full stack aplicando React en frontend y Spring Boot en backend para gestionar de forma segura los dispositivos del centro.

## 🚀 Guía de instalación

### Backend
- Abrir el proyecto en el IDE  
- Configurar base de datos en `application.properties`  
- Ejecutar Spring Boot  

### Frontend
- Instalar dependencias: npm install  
- Ejecutar proyecto: npm run dev

## 📱 Guía de uso

- Iniciar sesión con usuario registrado  
- Seleccionar fechas del préstamo  
- Solicitar portátil disponible  
- Devolver portátil al finalizar el uso  
- Consultar historial de préstamos (si está implementado)
