# 📌 Introducción

## Descripción del proyecto
Aplicación web para la gestión de préstamos de portátiles en un entorno educativo. Permite a los usuarios iniciar sesión, solicitar un portátil disponible, devolverlo y consultar su estado.

## Justificación
Este proyecto surge para digitalizar y automatizar la gestión manual de préstamos de dispositivos en centros educativos, mejorando el control y reduciendo errores.

## Objetivos
- Automatizar el préstamo de portátiles
- Controlar la disponibilidad de equipos
- Registrar historial de préstamos
- Implementar autenticación de usuarios

## Motivación
Desarrollar una aplicación full stack real aplicando React en frontend y Spring Boot en backend para un control más seguro de los materiales del centro.

# ⚙️ Funcionalidades y tecnologías

## Funcionalidades
- Login de usuario
- Solicitud de portátil disponible
- Asignación automática
- Devolución de portátil
- Historial de préstamos (PENDIENTE / EN DESARROLLO si falta pulir)
- Gestión de estados (DISPONIBLE / PRESTADO / DEVUELTO)

## Tecnologías utilizadas

### Frontend
- React
- JavaScript
- HTML / CSS

### Backend
- Java
- Spring Boot
- Spring Data JPA
- API REST

### Base de datos
- MySQL
